Release - Results from your long read (Nanopore) assembly

Project: Example project
Released by: RIC Analyst (cribioinfo@uic.edu)
Date: Fri, Jul 5, 03:16 PM,2019.

Results from an assembly using both long-read (Nanopore) and short-read (Illumina) sequence data.  The long-read data were used for the initial assembly and then the short-read data was used to error-correct (polish) the assembly.  In addition, the contigs were annotated using Prokka.
