Release - Results from your metagenomic assembly

Project: Example project
Released by: RIC Analyst (cribioinfo@uic.edu)
Date: Jan 11, 2019.

Results from a metagenomic co-assembly of all samples.   For the assembly all samples were combined into a single sequence dataset for assembly.  After assembly, the sequence data for each sample individually was mapped back to the contigs to determine read counts per sample.  In addition, the contigs were annotated using Prokka.
File Name File Type Description
report.html report Basic processing report
contigs_counts.txt result Raw sequence counts for each contig
contigs_data.txt result Details of assembled contigs
contigs_geneAnnotation.txt result Table of putative gene annotations for all contigs
contigs_KEGG_annotation.txt result Table of putative ortholog annotations for all contigs
metagenomic_assembly.summary.txt report Summary of metagenomic assembly
metagenomic_assembly.zip result Contigs from the assembly (zipped FASTA format)
orf_counts.txt result Raw read counts for predicted ORFs
